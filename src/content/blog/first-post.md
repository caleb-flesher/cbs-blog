# The first Caleb's Bike Shop post!

## March 7th, 2026

Welcome to the very first blog post for Caleb's Bike Shop (CBS)! This whole thing started as a fun little winter project and has already become so much more to me. In just the few short months since I've started the shop, I have already had so much fun with it! Between building The Barhopper and building this site, I have learned so much and I am excited to continue my learning journey.

Building the first bike and having gotten to ridden it a few times now, I have discovered that I still have much to learn for my first build. However, I have also learned that I have created a bike that is fun to ride, comfortable, and achieves what I orignally built it to do - get you from one place to another in a fun way.

Is the shop evolves, so too will this site. Be sure to check in regularly to see what's new!
